Array 관련 함수

uniq

배열 원소 중복 제거

uniq<T>(array: List<T> | null | undefined): T[];
// arg: array
// return: array

const siteList = [0, 1, 1, 2];
const uniqSiteList = uniq(siteList); // [0, 1, 2]

Collection 관련 함수

sortby

collection 값들을 원하는 필드를 기준으로 오름차순으로 정렬

sortBy(collection, [(iteratees = [_.identity])]);

//collection: 정렬 대상
//[iteratees=[_.identity]] 정렬 기준

// => return 정렬된 새로운 array
  • 여러 개의 필드값으로도 정렬 가능
sortBy(정렬을 원하는 컬렉션, [정렬 기준 필드값 1, 정렬 기준 필드값 2]);
  • 내림차 순으로 정렬하기 위해서는, reverse() 사용
sortBy(정렬을 원하는 컬렉션, 정렬 기준 필드값).reverse();

const users = [
  { user: "fred", age: 48 },
  { user: "barney", age: 36 },
  { user: "fred", age: 40 },
  { user: "barney", age: 34 },
];

const sortByFunc = sortBy(users, [
  function (o) {
    return o.user;
  },
]);
console.log(sortByFunc);
// => objects for [['barney', 36], ['barney', 34], ['fred', 48], ['fred', 40]]

const sortByTwoArg = sortBy(users, ["user", "age"]);
console.log(sortByTwoArg);
// => objects for [['barney', 34], ['barney', 36], ['fred', 40], ['fred', 48]]
