Vite + React + TypeScript 환경에서 Emotion Css Prop 사용하기

2024/1/21

Vite + React + TypeScript 환경에서 Emotion css prop를 사용하니 다음과 같이 계속 오류가 나는 것을 확인할 수 있었습니다.

검색해보니 TypeScript 말고도 Vite에서도 css prop를 사용하려면 추가 설정을 해줘야 하는 것을 알게 되어서 한번 정리해두려고 합니다.

TypeScript 설정

tsconfig.json

{
  "compilerOptions": {
    ...
    "jsx": "react-jsx",
    "jsxImportSource": "@emotion/react",
    ...
  },
  ...
}

[emotion 공식 문서] typescript

Vite 설정

vite.config.ts

import { defineConfig } from "vite";
import react from "@vitejs/plugin-react";

// https://vitejs.dev/config/
export default defineConfig({
  plugins: [
    react({
      jsxImportSource: "@emotion/react",
      babel: {
        plugins: ["@emotion/babel-plugin"],
      },
    }),
  ],
});
