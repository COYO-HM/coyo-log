Styled Components Global Style 설정하기

2022/6/24

createGlobalStyle 하나의 컴포넌트를 만들 수 있게 해주는데 랜더링이 될 때, 그 컴포넌트는 전연 스코프에 스타일들을 올려준다.

import { createGlobalStyle } from "styled-components";
import Router from "./Router";

const GlobalStyle = createGlobalStyle`
body {
  //global style 작성
}
`;

function App() {
  return (
    <>
      <GlobalStyle />
      <Router />
    </>
  );
}

export default App;
