yarn berry 적용하기

2024/1/24

yarn 을 이미 설치했다는 전제하에 진행하겠습니다. yarn 설치 여부는 yarn -v 을 쳐서 확인할 수 있습니다.

yarn berry로 전환하기

yarn set version berry

더욱 자세한 버전 변경 방법은 @yarnpkg/cli 을 참고해주세요.

아래 명령어를 입력하여 더이상 쓰지 않는 node_modules, package.lock.json 를 삭제해줍니다.

rm -rf node_modules
rm -rf package-lock.json

.yarnrc.yml 해당 파일에 nodeLinker 가 node-modules 라면 다음과 같이 변경해줍니다.

yarnPath: .yarn/releases/yarn-4.0.2.cjs
nodeLinker: pnp

이후에 설치를 진행해주면 완성입니다.

yarn install

Zero-installs 적용

🔗 Zero-installs 관련한 내용

  • yarn caching
  • 블로그 글

Zero-installs 적용시에

.gitignore

.yarn/*
!.yarn/releases
!.yarn/patches
!.yarn/plugins
!.yarn/sdks
!.yarn/versions
!.yarn/cache

Zero-installs 미적용시에

.gitignore

.pnp.*
.yarn/*
!.yarn/releases
!.yarn/patches
!.yarn/plugins
!.yarn/sdks
!.yarn/versions

💡 .gitignore 에서 해당 파일 앞에 !은 무시하지 말고 반드시 포함하라는 의미입니다.
