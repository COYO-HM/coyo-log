> Github Page
>
> github repository에 올린 `javaScript` 프로젝트를 배포하여 정적 웹 사이트를 무료로 호스팅할 수 있게 하는 서비스이다.

이전에 `react`로 만든 프로젝트를 한번 Github Page로 배포해보자.

# Project Github page 라이브러리 설치

```jsx
npm i gh-pages --dev
```

# Package.json 설정하기

`package.json` 에 다음과 같이 홈페이지 주소를 추가해준다.

```jsx
 "homepage": "https://사용자 아이디.github.io/레포지토리 이름/",
```

![homepage](/static/images/blog/github-page-publish-01.png)

배포를 위해 `deploy`를 추가해준다.

![deploy](/static/images/blog/github-page-publish-02.png)

다음 명령어를 입력해 배포해준다.

```bash
npm run deploy
```

# Github Page Repository에서 설정

- Repository 메뉴에서 Settings 메뉴로 들어가준다.

![Settings](/static/images/blog/github-page-publish-03.png)

- Settings 메뉴 중에 Pages 메뉴로 들어가면 Branch가 none으로 설정되어 있다.

![Pages](/static/images/blog/github-page-publish-04.png)

- 생성된 gh-pages 브랜치로 설정해준다.
