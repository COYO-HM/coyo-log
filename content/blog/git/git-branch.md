# 브랜치 생성

# 브랜치 이름 변경

```bash
git branch -m 이전이름 새로운 이름
```

# 브랜치 버전 확인

```bash
git branch -v
```

# 특정 브랜지에서 `pull` 받기

```bash
git pull origin 브랝치 이름
```

# 브랜치 삭제

```bash
git branch -d 브랜치 이름
```
