# API (Application Programming Interface)

> 어플리케이션 소프트웨어를 구축하고 통합하는 정의 및 프로토콜 세트로 때때로 API는 정보 제공자와 정보 사용자 간의 계약으로 지칭되며 소비자에게 필요한 콘텐츠(호출)과 생상자에게 필요한 콘텐츠(응답)를 구성한다.
