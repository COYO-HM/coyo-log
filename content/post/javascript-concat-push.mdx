# 1. Array.prototype.concat()

> `concat()`
메서드는 인자로 주어진 배열이나 값들을 기존 배열에 합쳐서 새 배열을 반환한다.
>

- 기존배열을 변경하지 않는다.
- 추가된 새로운 배열을 반환한다.
- `concat`은 메서드를 호출한 배열 뒤에 각 인수를 순서대로 붙여 새로운 배열을 생성한다.
인수가 배열이면 그 구성요소가 순서대로 붙고, 배열이 아니면, 인수 자체가 붙는다. 중첩 배열 내부로 재귀하지 않다.

    ```jsx
    const alpha = ['a', 'b', 'c'];
    const numeric = [1, 2, 3];

    alpha.concat(numeric);
    // 결과: ['a', 'b', 'c', 1, 2, 3]
    ```

    ```jsx
    const num1 = [1, 2, 3];
    const num2 = [4, 5, 6];
    const num3 = [7, 8, 9];

    num1.concat(num2, num3);
    // 결과: [1, 2, 3, 4, 5, 6, 7, 8, 9]
    ```

    ```jsx
    const alpha = ['a', 'b', 'c'];

    alpha.concat(1, [2, 3]);
    // 결과: ['a', 'b', 'c', 1, 2, 3]
    ```


# 2. Array.prototype.push()

> `push()`
메서드는 배열의 끝에 하나 이상의 요소를 추가하고, 배열의 새로운 길이를 반환힌다.
>

```jsx
var sports = ['축구', '야구'];
var total = sports.push('미식축구', '수영');

console.log(sports); // ['축구', '야구', '미식축구', '수영']
console.log(total);  // 4
```
