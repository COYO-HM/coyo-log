> Github Page
>
> github repository에 올린 `javaScript` 프로젝트를 배포하여 정적 웹 사이트를 무료로 호스팅할 수 있게 하는 서비스이다.

이전에 `react`로 만든 프로젝트를 한번 Github Page로 배포해보자.

# Project Github page 라이브러리 설치

```jsx
npm i gh-pages --dev
```

# Package.json 설정하기

`package.json` 에 다음과 같이 홈페이지 주소를 추가해준다.

```jsx
 "homepage": "https://사용자 아이디.github.io/레포지토리 이름/",
```

![homepage](https://user-images.githubusercontent.com/56423604/272323342-5973abc2-ae9c-4d22-b201-fa1a1670c876.png)

배포를 위해 `deploy`를 추가해준다.

![deploy](https://user-images.githubusercontent.com/56423604/272323504-ba7b0ddc-1993-48a9-b13e-5ca793903ab5.png)

다음 명령어를 입력해 배포해준다.

```bash
npm run deploy
```
