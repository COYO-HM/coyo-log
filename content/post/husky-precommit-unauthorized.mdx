# ⚠️ 문제

블로그를 배포하던 중에 다음과 같은 오류가 발생하였습니다.

![husky-precommit-unauthorized](https://github.com/COYO-HM/COYO-HM.github.io/assets/56423604/1407d749-eac6-46b6-8f2a-89ea956bc6da)

# 🔧 해결

찾아보니까 이는 권한 문제로 권한 문제로 .husky/pre-commit에 권한 설정을 해주면 간단하게 해결됩니다.

`husky/pre-commit` 을 잠깐 삭제해주었다가 다시 생성했는데 그로 인해 발생된 것 같습니다.

```bash
chmod +x .husky/pre-commit
```
